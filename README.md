## Install
- Install nodejs
- `npm install hexo`

Checkout `package.json` for npm commands

## build
`npm run build`

## start local
`npm run start`

## deploy
`hexo deploy` or `npm run deploy`

### build && deploy
`npm run build && hexo deploy`

### git update
`git add --all && git commit -am "Update" && git push` or `npm run push`

### shortcuts
`npm run deploy` \
`npm run push`

### new post
`hexo new post <this_is_post_title>`
